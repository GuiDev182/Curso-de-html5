Um código em linguagem C

Usando // comenta-se apenas a linha corrente a partir de onde o // foi colocado.

Já usando /* */, o texto que fica entre /* e */ torna-se um comentário, mesmo que esteja contido em mais de uma linha.

    
     main()
     {
     // Isto é um comentário 

     /* Isto também é um comentário
     Continua sendo comentário aqui */
     }
